# 🚀 رائد الرقمي - موقع تسويق إلكتروني احترافي

## 📋 وصف المشروع
موقع شخصي احترافي لخبير التسويق الإلكتروني يعرض خدمات التسويق على المنصات، إنشاء المتاجر الإلكترونية، وتطوير التطبيقات. مصمم بأسلوب رسمي واحترافي لتحويل الزوار إلى عملاء محتملين.

---

## ✅ الميزات المكتملة

### الصفحات والأقسام:
- **Hero Section**: قسم رئيسي مميز مع انيميشن، بطاقات عائمة، وجزيئات متحركة
- **About Section**: قسم "من أنا" مع أشرطة المهارات المتحركة
- **Services Section**: 6 خدمات احترافية مع تأثيرات hover ثلاثية الأبعاد
- **Stats Section**: عداد أرقام متحرك (250+ عميل، 380+ مشروع...)
- **Portfolio Section**: معرض أعمال مع فلتر تفاعلي (تسويق / متاجر / تطبيقات)
- **Process Section**: خطوات طريقة العمل (4 خطوات)
- **Testimonials Section**: سلايدر آراء العملاء مع تحكم ودعم اللمس
- **Contact Section**: نموذج تواصل كامل متصل بقاعدة البيانات
- **Footer**: فوتر احترافي مع روابط وبيانات التواصل

## 🗄️ قاعدة البيانات

### جدول: `contact_messages`
| الحقل | النوع | الوصف |
|-------|-------|-------|
| id | text | معرف فريد |
| name | text | اسم العميل |
| email | text | البريد الإلكتروني |
| phone | text | رقم الهاتف |
| service | text | الخدمة المطلوبة |
| budget | text | الميزانية التقريبية |
| message | rich_text | تفاصيل الرسالة |
| status | text | حالة الرسالة (new/read/replied) |
| submitted_at | datetime | وقت الإرسال |

## ✏️ التخصيص المطلوب من صاحب الموقع

يجب تعديل هذه البيانات في ملف `index.html`:

1. **الاسم**: استبدل "رائد الرقمي" باسمك الحقيقي أو العلامة التجارية
2. **رقم الهاتف**: غيّر `+966 5XX XXX XXXX` برقمك الحقيقي
3. **البريد الإلكتروني**: غيّر `info@yourmail.com` ببريدك
4. **رابط واتساب**: في زر الواتساب العائم، غيّر `966XXXXXXXXX` برقمك
5. **روابط السوشيال ميديا**: أضف الروابط الحقيقية لحساباتك
6. **صورتك الشخصية**: استبدل أيقونات المستخدم بصورتك
7. **أرقام الإحصائيات**: عدّل الأرقام لتعكس إنجازاتك الحقيقية
8. **الأعمال السابقة**: عدّل بطاقات Portfolio بمشاريعك الحقيقية
